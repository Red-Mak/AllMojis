# AllMojis
An iOS keyboard to see how emojis will look like in others platforms/apps.
This Application was rejected by Apple twice, so i open source it :)

Plateforms/Applications supported are:
 * apple
 * emoji-one
 * facebook
 * google
 * lg
 * htc
 * messenger
 * microsoft
 * samsung
 * twitter
